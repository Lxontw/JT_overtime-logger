# 更新日誌 (Changelog)

所有對 **加班記錄器** 的重要變動都將記錄在此。

## [1.1.0] - 2026-01-24

### 修正 (Fixed)
- **修正跨日日期錯誤**：解決了在凌晨（如 03:00）開啟網頁時，日期因時區轉換（UTC）而自動跳回前一天的 Bug。
- **改進初始化邏輯**：棄用 `valueAsDate`，改用在地化字串格式 (`YYYY-MM-DD`)，確保「所見即所得」，系統不再自作主張扣除日期。
- **設定頁面切換**：修正了第一次點擊齒輪圖標時，設定頁面可能出現空白或無法正確切換的問題。
- **標題重疊問題**：重構頂部佈局，將功能圖示移至按鈕上方，徹底解決標題與圖示在窄螢幕下的重疊現象。
- **欄位間距連動**：改用 `rem` 相對單位定義間距，確保字體放大時，欄位間的距離能自動拉開，避免手指誤觸。
- **OCR 佈局對齊**：優化「加班事由」與 OCR 按鈕的排列方式，確保在小螢幕下輸入框仍能維持 100% 寬度。

### 新增 (Added)
- **介面外觀升級**：專案更名為「加班記錄器」，並採用更現代化的 UI 圓角與陰影設計。
- **字體放大功能 (AA)**：新增 iOS 風格的「AA」圖示，提供三段式字體大小循環切換，專為長輩與老花眼需求設計。
- **深色模式 (Dark Mode)**：新增 🌓 圖示切換功能，支援淺色與深色模式轉換，提升低光環境下的使用舒適度。
- **個人姓名設定**：新增「您的姓名」欄位，儲存於本地瀏覽器 (LocalStorage)，送出登記時會自動帶入。
- **自訂常用事由**：
  - 現在可以在設定頁面手動輸入常用的加班理由（以逗號分隔）。
  - 主畫面新增「快速選擇」下拉選單，點選後自動填入事由框。
- **預設事由清單**：加入預設值（交接輪值、百貨類處理、教科書處理、數位印刷、臨時急件）。

### 變更 (Changed)
- **圖示位置調整**：將設定、深色模式、字體切換圖示由頂部移至「送出登記」與「儲存並返回」按鈕的正上方，方便拇指點擊。
- **結束時間預填**：現在開啟網頁時，結束時間會自動對齊目前的系統時間，方便晚班下班直接登記。
- **程式碼註解**：在 JavaScript 關鍵邏輯處補上繁體中文說明，提升後續維護便利性。
